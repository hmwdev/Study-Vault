# Study-Vault

A structured repository for my courses, practice, exercises, notes, and self-learning work.

This is where I keep the code I write while learning and building across:

- Python
- Rust
- Go
- React
- FastAPI
- Data Structures and Algorithms
- and other topics I study over time

The goal is simple: **learn well, stay organized, and keep a clear record of progress**.

---

## Why this repo exists

Learning gets messy fast.

Notes end up in one place. Code ends up in another. Small experiments get lost. Good ideas disappear.

I made **Study-Vault** to keep that from happening.

This repo is my personal system for:

- saving course code
- practicing concepts
- solving exercises
- keeping small experiments
- revisiting ideas later
- tracking growth over time

It is both a **working archive** and a **learning log**.
