# 🐍 Python Mastery: Foundations to Advanced
### Inspired by Abdul Bari's Comprehensive Python Course

This directory serves as a structured log of my journey through advanced Python programming. My goal is to build a "mental library" of algorithms, data structures, and Pythonic design patterns.

---

## 🎯 Learning Objectives
*   Master **Memory Management** and the Python Virtual Machine (PVM).
*   Implement complex **Data Structures** (Linked Lists, Trees, Graphs) from scratch.
*   Understand **Algorithm Analysis** (Big O Notation) within a Pythonic context.
*   Deep dive into **Object-Oriented Programming (OOP)**.

## 📂 Structure
- `01-Basics/`: Data types, control flow, and memory references.
- `02-Recursion/`: Solving mathematical and logical problems using recursive calls.
- `03-Data-Structures/`: Custom implementations of stacks, queues, and trees.
- `04-OOP-Deep-Dive/`: Classes, inheritance, polymorphism, and decorators.

## 💡 Beyond the Course
In addition to course exercises, I include a `my_logic/` folder in each section where I apply the learned concepts to unique scenarios, such as:
*   **Statistical Logic:** Applying Python to probability problems.
*   **Performance Testing:** Measuring execution time for different algorithmic approaches.
